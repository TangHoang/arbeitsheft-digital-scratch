# Digitales Arbeitsheft zur Einführung in die Programmierung mit Scratch und Whisker

## Lokale Ausführung des digitalen Arbeitshefts

Dieses Repository enthält den Quellcode für das im Rahmen der Masterarbeit entwickelte digitale Arbeitsheft. Falls die Bereitstellung über GitHub Pages nicht verfügbar sein sollte, kann die Anwendung mit den folgenden Schritten lokal installiert und ausgeführt werden.

## Voraussetzungen

Für die Ausführung wird eine installierte Node.js-Umgebung (Version 16.0 oder höher) sowie der Paketmanager npm benötigt.

## Installation

1. Repository klonen:
   git clone https://github.com/TangHoang/arbeitsheft-digital-scratch

2. In das Projektverzeichnis wechseln:
   cd [DEIN_ORDNERNAME]

3. Abhängigkeiten installieren:
   npm install

## Ausführung im Entwicklungsmodus

Um die Anwendung lokal zu starten, führen Sie den folgenden Befehl aus:

npm run dev

Die Anwendung ist anschließend standardmäßig unter http://localhost:5173 (oder der im Terminal ausgegebenen Adresse) erreichbar.


## Technische Hinweise

- Internetverbindung: Eine aktive Verbindung ist erforderlich, um die Kommunikation mit den Servern der Universität Passau für das Whisker-Test-Framework zu ermöglichen.
